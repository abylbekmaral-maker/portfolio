포트폴리오

"술이 어때?" – 음료 추천 앱 애플리케이션

프로젝트 설명

사용자가 선호하는 주종과 분위기에 맞춰 칵테일·주류 레시피를 추천해주는 모바일 앱 애플리케이션입니다. 메뉴별로 기존 레시피를 조회하고, 새로운 레시피를 추가·수정·삭제할 수 있도록 설계했습니다. 직관적인 화면 구성과 명확한 카테고리 구분을 통해 사용자가 빠르게 원하는 정보를 찾을 수 있도록 UX를 고민했습니다.

주요 구현 내용

  • 레시피 데이터를 저장·조회·수정·삭제하는 기능 개발
  • 로그인/회원가입, 아이디·비밀번호 찾기 등 기본 회원 기능 구현
  • 실제 사용 흐름을 고려하여 메인 화면, 상세 화면, 관리 화면 간의 UX 설계

주요 화면 구성

1. 주요 메뉴 화면 (Main Menu)

  • 상단에 ‘술이 어때’ 타이틀과 로그인/회원가입 버튼 배치
  • 하단에 위스키·보드카·데킬라·기타 4가지 주류 카테고리 카드로 구분
  • 서비스 소개 이미지를 통해 “편리한 주류 소개/추천 앱”이라는 콘셉트 전달

2. 로그인 화면 (User Login)

  • 개인화된 서비스 제공을 위한 필수 진입 화면
  • 중앙에 ID/비밀번호 입력 필드와 ‘로그인’ 버튼을 배치해 시선 집중
  • 아이디 찾기, 비밀번호 찾기, 회원가입 버튼으로 사용자 편의성 강화

3. 레시피 상세 화면 (Cocktail Recipe Detail)

  • 선택한 칵테일(예: 올드패션드)에 대한 상세 정보(이름, 도수, 소개, 재료)를 제공
  • IBA 레시피, 조주기능사 레시피 등 기준별 레시피 탭 구성
  • 제조 순서를 단계별로 안내해 사용자가 그대로 따라 만들 수 있도록 설계

Network Lab – Packet Tracer 네트워크 실습

도구: Cisco Packet Tracer · 내용: 라우터/스위치 구성, 서브넷팅, SSH 설정 등

CCNA 과정과 전공 수업을 기반으로, 실제 네트워크 구성을 Packet Tracer로 설계하고 라우터와 스위치, 서버, PC를 연결하여 통신이 가능한 구조를 구성했습니다. CLI 환경에서 라우터를 설정하며, 네트워크 장비의 동작 원리를 직접 체험할 수 있었습니다.


  • 서브넷팅을 통해 네트워크를 나누고 IP 주소를 직접 설계
  • 라우터 간 라우팅 설정 및 기본 보안 설정 실습
  • SSH 접속 설정을 통해 원격 관리 환경 구성
  • 문제가 발생했을 때 Ping, Traceroute 등을 활용해 원인 추적
