도전을 즐기는 유학생 출신 개발자

저는 키르기스스탄 오시(Osh)에서 태어나 한국으로 유학와 한국어를 처음부터 다시 배우며 컴퓨터정보과를 전공했습니다. 새로운 언어와 환경에 적응하는 과정에서 끈기와 문제 해결력을 키웠고, 지금은 한국어를 포함해 4개 국어로 소통하며 다양한 배경의 사람들과 함께 일하는 것에 큰 즐거움을 느낍니다.

학교에서는 Java 기반 객체지향 프로그래밍, 데이터베이스, 네트워크, 운영체제 등을 공부했고, 팀 프로젝트로 실제 사용자를 고려한 모바일 웹 서비스를 구현해보며 UI 흐름 설계와 사용자 경험에 특히 흥미를 느꼈습니다. 사용자가 편하게 느끼는 서비스, 그리고 팀과 함께 성장하는 개발자가 되고 싶습니다.
